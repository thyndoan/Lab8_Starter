**Q**: How are graceful degradation and service workers related?

**A**: Graceful degradation means start with full features and handle failures gracefully. In this case, service workers allow the website to fall back to cached content when there is no internet. So instead of the whole website completely blank, it shows meaningful content (even when the content is not up to date)
